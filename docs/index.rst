**pushjet**
===========

.. code-block:: python
    :emphasize-lines: 3

    import pushjet
    service = pushjet.Service.create("Python Pushjet API News")
    service.send("This is a pretty sweet module!")

**pushjet** is the name of a Python module for interfacing with `Pushjet <https://pushjet.io/>`_. Simple and snazzy name, eh? Take a look at the examples and get a running start.
